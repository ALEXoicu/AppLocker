# AppLocker
This is a simple application to lock any .exe file with a password.

Any time you try to open the .exe after locking it will ask you for a password.

Usage:
- Locking:
  - Open locker.exe
  - Press "Lock File"
  - Select your .exe file from the file picker
- Unlocking:
  - Open locker.exe
  - Press "Unlock File"
  - Select your .exe.locked file from the file picker

!!WARNINGS!!:
- Do not delete the "launcher_template.exe" file and make sure its in the same folder with "locker.exe"!
- Deleting the .exe.locked file will cause you to lose your .exe file!
